## Descripción
Trabajo práctico nº1. Curriculum Web - Carballo Mario.  
El objetivo es construir un CV profesional digital utilizando **HTML5, CSS3, JavaScript** y desplegarlo en **GitHub Pages**.

---

## Requisitos cumplidos
- Encabezado con foto, nombre y título.  
- Sección "Sobre mí".  
- Experiencia ficticia con link y email.  
- Educación.  
- Habilidades con barras de progreso.  
- Idiomas.  
- Contacto con correo y GitHub.  
- Formulario de contacto.  
- Fuente de Google Fonts y variable CSS.  
- Botón con acción en JavaScript.  
- Publicación en GitHub Pages.
